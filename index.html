Personal History Timeline

A timeline application for managing family memories and special events

By Robert Bascom dba parsemonkey

Key Features

  • Year-to-year scrolling timeline interface
  • Complete CRUD support for event management
  • Comprehensive search and data management tools
  • Bulk data entry and batch processing updates
  • Import/Export support for CSV files
  • Duplicates removal functionality
  • Modest theme support for personalization
  • Includes test database with 1,000 sample records

Trial Version Information

This is trial software. This version provides a 21-day evaluation period for testing and evaluation purposes. After the trial period, a one-time license fee of $38 is required for continued use.

Built with Python3, SQLite, and PySide6 for Linux systems.
